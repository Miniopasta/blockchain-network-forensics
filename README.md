## Deskripsi Tugas

Tugas ini bertujuan melakukan akuisisi bukti jaringan menggunakan file PCAP, menghitung nilai hash SHA-256 sebagai identitas digital bukti, serta menyimpan metadata bukti ke dalam blockchain sederhana menggunakan Python.

## Tools yang Digunakan

* Python 3.11
* Scapy
* Wireshark
* Visual Studio Code

## Struktur Program

Program terdiri dari:

1. Akuisisi data jaringan (PCAP)
2. Perhitungan hash SHA-256
3. Pembuatan blockchain sederhana
4. Validasi integritas blockchain

## Cara Menjalankan

```bash
pip install scapy
python blockchain_pcap.py
```
